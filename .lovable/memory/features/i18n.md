name: Internationalization
description: i18n system with EN/ES/PT/FR, globe toggle, auto-detect browser language
type: feature
---
- I18nContext at `src/i18n/I18nContext.tsx` with `useI18n()` hook
- Translation files: `src/i18n/{en,es,pt,fr}.ts` implementing `Translations` interface
- Locale stored in `localStorage` key `dtt_locale`, auto-detects browser language
- `LanguageToggle` component renders Globe icon dropdown at top-right
- Integrated into: AgeVerification, BottomNav, LegalPages, GlobalSearch, Index profile tab
- All legal compliance text (age gate, ToS, privacy) fully translated
- Prices remain in USD ($21/$101) — fixed Bit-Token value
